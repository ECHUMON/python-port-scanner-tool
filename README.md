# 🔐 Cybersecurity Port Scanner (Python + Streamlit)

A lightweight, multi-threaded TCP Port Scanner built using Python and Streamlit.  
This tool helps identify open ports and associated services on a target IP address or domain for educational cybersecurity and network reconnaissance purposes.

---

## 🚀 Features

- 🌐 Scan IP addresses or domain names
- ⚡ Multi-threaded scanning for fast results
- 🔎 Detect open and closed TCP ports
- 🧠 Common service identification (SSH, HTTP, FTP, MySQL, etc.)
- 📊 Clean interactive Streamlit dashboard
- ⏱ Scan duration tracking
- 📁 Export scan results as CSV report
- 🎨 Modern UI with custom background styling

---

## 🧠 How It Works

- The user enters a target IP or domain
- Domain is resolved into an IP address
- Selected ports are scanned using TCP connection attempts
- Results are categorized as OPEN or CLOSED
- Output is displayed in a structured table and downloadable as CSV
